# REQ-COST.md

## Summary [REQ-COST-SUMMARY]

This document defines requirements for token accounting, cost estimation, and spend visibility.

### REQ-300 — Per-Interaction Cost
Weave shall record token usage and cost for each provider interaction.

### REQ-301 — Session Cost Visibility
Weave shall provide per-session cost visibility to the builder.

### REQ-302 — Estimated Budget Awareness
Weave shall estimate request size before submission when provider support or calibrated estimation is available.

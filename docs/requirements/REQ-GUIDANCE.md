## Summary [REQ-GUIDANCE-SUMMARY]

This document defines requirements for builder-visible reasoning, workflow guidance, and transparency.

### REQ-500 — Explain Routing
Weave shall expose why a model or workflow path was selected.

### REQ-501 — Surface Conflicts
Weave shall detect and surface conflicts between requests and project decisions or constraints.

### REQ-502 — Encourage Acceptance Criteria
Weave shall support bounded work by reinforcing acceptance criteria before implementation work begins.
